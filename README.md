# HR-Analytics-Dashboard-project-
This project was developed using Microsoft Excel for data preparation and Power BI for data modeling, visualization, and reporting.The HR Analytics Dashboard is an interactive business intelligence solution designed to help HR professionals analyze workforce data, monitor employee trends, and make data-driven decisions
## Project Objective
Analyze employee demographics and workforce distribution.
Monitor employee attrition and retention rates.
Track department-wise employee performance.
Identify trends in hiring, promotions, and employee turnover.
Support strategic HR decision-making through data visualization.
